# Anti-Patrones de Copy: Lo Que NUNCA Escribir

Esta lista existe porque estas palabras y frases se han usado tanto que perdieron todo significado. Cuando las ves, tu cerebro las ignora. Peor aún: generan desconfianza.

## Palabras Prohibidas

### Buzzwords Vacías

| Palabra/Frase | Por qué evitarla | Alternativa |
|---------------|------------------|-------------|
| Revolucionario | Nada es revolucionario. Si lo fuera, no necesitarías decirlo | Describe qué hace diferente |
| Disruptivo | Usado por todos, significa nada | Explica el cambio concreto |
| Game-changer | Hipérbole que nadie cree | Muestra el resultado |
| Innovador | Auto-proclamarse innovador es sospechoso | Deja que el producto hable |
| De última generación | ¿Generación de qué? | Especifica la tecnología |
| Cutting-edge / State-of-the-art | Jerga de brochure corporativo | Describe la capacidad real |
| Sinergia | Palabra de consultor de los 90s | Di colaboración o integración |
| Ecosistema | A menos que sea literal | Sistema, plataforma, o nada |
| Holístico | Vago y pretencioso | Completo, integral (si es necesario) |
| Robusto | No dice nada útil | Especifica qué lo hace confiable |

### Verbos Inflados

| Verbo | Problema | Alternativa |
|-------|----------|-------------|
| Potenciar | ¿Potenciar cómo? | Mejora X en Y% |
| Maximizar | Vago | Aumenta, duplica (con datos) |
| Optimizar | Sobreusado | Reduce, acelera, simplifica |
| Transformar | Promesa enorme | Cambia, mejora, automatiza |
| Revolucionar | Ver arriba | Describe el cambio |
| Empoderar | Corporativo y vacío | Permite, da acceso a |
| Apalancar | Traducción literal horrible | Usar, aprovechar |
| Escalar | Solo si hablas de infraestructura real | Crecer, expandir |

### Frases Cliché

| Frase | Por qué falla |
|-------|---------------|
| "Lleva tu X al siguiente nivel" | ¿Qué nivel? Es vago y gastado |
| "Solución integral" | Todas las soluciones se dicen integrales |
| "Todo en uno" | Demasiado genérico para ser creíble |
| "Hecho para ti" | Personalización falsa |
| "Más que un X, somos Y" | Fórmula agotada |
| "El futuro de X" | Arrogante y vacío |
| "X reimaginado" | Cliché de Silicon Valley |
| "La forma inteligente de X" | Implica que el usuario es tonto |
| "Nunca fue tan fácil" | Promesa que raramente es cierta |
| "Con solo un clic" | Casi nunca es literalmente un clic |

## Patrones Problemáticos

### 1. Superlativos No Verificables

**Malo:**
- "La mejor herramienta de gestión"
- "El software más completo del mercado"
- "Líder en innovación"

**Por qué:** Si no puedes probarlo, no lo digas. Además, Google puede penalizarte por claims no verificables.

**Mejor:** Deja que las métricas y testimonios hablen por ti.

### 2. Promesas Imposibles de Cumplir

**Malo:**
- "Duplica tus ventas"
- "Resultados garantizados"
- "Transforma tu negocio en 30 días"

**Por qué:** Creas expectativas que no puedes cumplir. El usuario se siente estafado.

**Mejor:** Promesas específicas y alcanzables: "Nuestros usuarios ahorran un promedio de 5 horas semanales"

### 3. Urgencia Artificial

**Malo:**
- "¡Oferta por tiempo limitado!" (siempre activa)
- "Solo quedan 3 plazas" (siempre quedan 3)
- "Precio sube mañana" (nunca sube)

**Por qué:** Dark pattern que destruye confianza. El usuario lo nota.

**Mejor:** Si hay urgencia real, explícala. Si no, no la inventes.

### 4. Jerga Técnica Innecesaria

**Malo:**
- "Arquitectura cloud-native con microservicios containerizados"
- "Leveraging machine learning algorithms"
- "Blockchain-enabled decentralized solution"

**Por qué:** Si tu audiencia no es técnica, los alejas. Si es técnica, parece que compensas con buzzwords.

**Mejor:** Explica qué significa para el usuario: "Se actualiza automáticamente, sin que tengas que instalar nada"

### 5. Vaguedad Estratégica

**Malo:**
- "Mejora tu productividad" (¿cómo? ¿cuánto?)
- "Ahorra tiempo" (¿cuánto tiempo?)
- "Resultados reales" (¿qué resultados?)

**Por qué:** Si no puedes ser específico, el usuario asume que no tienes nada real que decir.

**Mejor:** "Reduce el tiempo de onboarding de 2 semanas a 2 días"

## Test de Honestidad

Antes de publicar cualquier copy, pásalo por estas preguntas:

1. **¿Podría defenderlo en un juicio?** - Si un abogado te preguntara por evidencia de cada claim, ¿la tienes?

2. **¿Mi competidor podría usar este mismo texto?** - Si sí, es demasiado genérico.

3. **¿Un usuario real hablaría así?** - Lee el copy en voz alta. Si suena a robot de marketing, reescríbelo.

4. **¿Qué pasa si un usuario escéptico lo lee?** - ¿Se reiría? ¿Pondría los ojos en blanco?

5. **¿Estoy describiendo el producto o mis deseos?** - El copy debe reflejar lo que el producto hace, no lo que quisieras que hiciera.

## Nota Final

El objetivo no es ser aburrido o no-persuasivo. Es ser **persuasivo con sustancia**.

Un beneficio real, bien articulado, siempre gana a diez buzzwords juntas. La honestidad no es una limitación creativa—es un filtro que fuerza mejor copy.
